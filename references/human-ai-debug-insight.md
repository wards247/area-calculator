# 人类直觉 vs AI推理：Debug 中的信息传递不对称

> 记录于 2026-07-01，A24地块面积计算器的"大陆板块漂移"bug 修复过程。

---

## 现象

描边完成后，红色描边线正确，但蓝色格子整体向右偏移，与底图错开。按 F12 打开开发者工具后自动归位。

## 两轮 Debug 的差异

### 第一轮：AI 盲摸

**输入**（用户）："如图，格子往右边漂移了，和图层错开了"

**AI 的反应**：
- 检查 `imgToDisp` 坐标转换
- 检查 `scale` 缩放因子
- 检查 `fillRect` 的 `+0.5` 偏移
- 检查 `getBoundingClientRect` 浮点数精度
- 检查 canvas `width/height` 与 CSS `width/height` 的对齐
- 检查 `object-fit: contain` 的渲染行为
- 检查 `offsetX` 是否有小数误差
- …反复排查近 15 处代码，均未命中

**耗时**：大量推理轮次，零进展。

### 第二轮：人类给因果链

**输入**（用户）："一面板弹出来把地图往左推了，但格子没跟；按 F12 就恢复"

**AI 的反应**：
- 立即定位：`ratioPanel.style.display = 'flex'` → `map-area` 宽度缩小 200px
- canvas drawing buffer 未更新 → 坐标系错位
- F12 触发 `window.resize` → `resize()` → 修好

**耗时**：一行 `requestAnimationFrame(resize)`，秒修。

---

## 根因分析

不是 AI 不行，也不是人不会描述。而是**信息形态的根本不对称**：

| | 人类 | AI |
|---|---|---|
| **感知能力** | 看到界面、时间线、因果 | 只看到代码和静态截图 |
| **输入习惯** | 给结论（"漂移了"） | 需要因果链（"X变了导致Y"） |
| **推理方式** | 直觉跳跃，瞬间锁定 | 穷举排查，逐项验证 |

人类天然会跳步骤——眼睛看到全过程，"大陆板块漂移"在脑子里已经是完整的因果故事。但给 AI 的只有最后一帧画面。AI 拿着最后一帧，只能倒推出所有可能的原因。

---

## 启发

**给 AI 的信息，不是越精确越好，而是越"前端"越好。**

不是："格子偏移了，帮我看看哪里错了"（结论）

而是："我点完描边后，右侧弹出一个面板，然后图片往左缩了，但格子和红框没跟过去；我按 F12 打开开发者工具后它们又自动对齐了"（因果链 + 关键触发条件）

往前多描述几步你看到的因果，比给十张截图都管用。

---

## 类比

你是侦探在案发现场，你有眼睛、有直觉、能看到凶手的脚印。

AI 是法医在实验室，只有你寄来的物证。

你寄来的是一根头发，AI 能分析出 DNA。但你如果写一句"死者是他杀，邻居听到争吵声，厨房有血迹"，AI 能直接锁定嫌疑人。
